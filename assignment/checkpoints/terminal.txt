# 8 - Why would you run the command 'git status'?
The git status displays the state of the working directory. It also shows the files that have not been tracked. It lets you know the changes that have been made and also the changes that have not been made.

git add . will add everything to git that is in the current folder. once you use git add . the files will no longer be untracked, and they will change to be committed. in addition the commit is what saves the version of the folder.
